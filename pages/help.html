Project Scope

The usable pages of this project are:
  • Register
  • Login (index)
  • Home
  • Tasks

Any other pages that seem accessible via a button have NOT been implemented as they are part of the whole system and this project focuses ONLY on the main features of the task manager.

How to use available features

Add a task

On the sidebar on the left of the screen, go to "Tasks". Find "Create a task" and click "Create". Finally, follow the instructions in the pop-up window.

View a task

On the sidebar on the left of the screen, go to "Tasks". Find a task to view in "Deadlines" (under one of the priority lists) and click on the name of the task to reveal a window.

Edit a task

On the sidebar on the left of the screen, go to "Tasks". Find "Edit a task" and click "Edit". Choose a task to edit in the follow up window. Finally, follow the instructions in the pop-up window.

Delete a task (Ongoing and Completed)

On the sidebar on the left of the screen, go to "Tasks". Find a task to delete in the "Deadlines" or "Completed Tasks" list. Click on the trash icon besides the task item.

Clear all tasks

On the sidebar on the left of the screen, go to "Tasks". Find "Clear all tasks" and click "Clear".
